# 贡献指南

感谢您对斗地主游戏项目的兴趣！我们欢迎各种形式的贡献。

## 如何贡献

### 报告Bug
- 使用GitHub Issues提交bug报告
- 详细描述问题的现象、复现步骤和预期行为
- 附上截图或视频演示（如果可能）

### 功能建议
- 通过Issues提出新功能建议
- 清晰地描述功能的目的和预期行为
- 提供可能的实现方案

### 提交Pull Request
1. Fork项目
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启Pull Request

## 代码规范

- 遵循Qt/C++编码规范
- 使用有意义的变量名和函数名
- 添加必要的注释和文档
- 在提交前测试您的代码

## 联系方式

如有任何疑问，欢迎通过Issues联系我们。
