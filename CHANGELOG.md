# 更新日志 (CHANGELOG)

所有项目的显著变化都记录在此文件中。

## 项目背景

本项目是深圳香港中文大学（CUHKSZ）**CSC3002 C/C++ Programming** 课程的小组作业。

### 📦 系统要求

- **Windows**: Windows 10/11 (64-bit)
- **Linux**: Ubuntu 18.04+ / Debian 10+
- **macOS**: macOS 10.13+
- **内存**: 2GB RAM 最低
- **磁盘**: 200MB 空间

### 🔧 技术栈

- **Frontend**: Qt 5.15.2, C++11
- **AI**: DouZero (PyTorch based)
- **Build**: qmake
- **License**: LGPL v3

### 📁 项目结构

核心模块：
- GameControl - 游戏主控制器
- GamePanel - 游戏显示面板
- Card/PlayHand - 卡牌和手型管理
- Player/Robot - 玩家和 AI 实现
- UI Components - 各种界面组件

代码行数：
- C++ 代码: 3000+ 行
- Qt UI 设计: 500+ 行
- 文档: 2000+ 行
